Mi proyecto de E-commerce

Este es un sitio web de comercio electrónico sencillo, diseñado para exhibir mis habilidades en HTML y CSS. El e-commerce simula una tienda en línea, donde los usuarios pueden explorar diferentes categorías de productos y ver detalles específicos de cada uno. En su estructura, he utilizado una maquetación responsiva con HTML semántico y estilos personalizados en CSS, garantizando una experiencia visual atractiva y adaptada a dispositivos móviles.

Cartelera de cine

Presenta una lista de películas en cartelera con un diseño atractivo y funcional. Utilizando HTML para la estructura y CSS para el estilo, esta cartelera incluye detalles como títulos de películas, horarios de función y sinopsis, organizados de manera clara y visualmente agradable. El objetivo de este proyecto es mostrar la capacidad para crear interfaces web dinámicas y bien organizadas, ideales para una experiencia de usuario intuitiva y atractiva.
